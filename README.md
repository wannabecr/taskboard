# TaskBoard by NullYex

A full-stack assignment tracker built with Vanilla JS, TailwindCSS, Express.js, and Supabase.

## Features
- Task management (Add, Edit, Complete)
- Google Authentication & Email/Password Sign-in
- File uploads for questions and answers
- Custom UI theming

## How to Run Locally

1. Clone the repository
2. Install dependencies: `npm install`
3. Rename `.env.example` to `.env` and add your Supabase credentials.
4. Start the server: `npm start`
5. Open `http://localhost:3000` in your browser.
